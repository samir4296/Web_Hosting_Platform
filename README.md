# Web Hosting Platform

A simple and modern web hosting platform built with **Node.js** and **JavaScript**. This project allows users to host, manage, and deploy their static websites easily.

## ✨ Features

- Easy web hosting for static websites
- User-friendly interface for managing hosted sites
- Automated deployment support
- Built with modern JavaScript
- End-to-end testing with Playwright

## 🛠️ Technologies Used

- **Node.js**
- **JavaScript** (100%)
- **Playwright** (for automated testing)

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)
